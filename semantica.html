Definicion y ejemplo de las nuevas etiquetas HTML5

1. Header (Encabezado):

Se utiliza para agrupar enlaces de navegación, como menús de navegación o barras de menú.

Ejemplo:

    La barra de menú en la parte superior de un sitio web con enlaces a "Inicio", "Acerca de nosotros", "Contacto", etc.

2. Footer (Pie de página):

El pie de página de una página o de una sección. Usado para información de copyright, enlaces a términos de servicio, información de contacto, etc.

Ejemplo:

    La parte inferior de una página que tiene el año de copyright y enlaces a políticas de privacidad.

3. Nav (Navegación):

Se utiliza para agrupar enlaces de navegación, como menús de navegación o barras de menú.

Ejemplo:

    La barra de menú en la parte superior de un sitio web con enlaces a "Inicio", "Acerca de nosotros", "Contacto", etc.

4. Main (Principal):

Marca la sección principal del contenido de una página. No debe usarse más de una vez en una página.

Ejemplo:

    La parte central de una página que contiene el contenido principal, como artículos o información clave.

5. Section (Sección):

Utilizada para dividir el contenido en secciones temáticas o grupos relacionados de información.

Ejemplo:

    Una página de noticias puede tener secciones para "Política", "Deportes" y "Entretenimiento".

6. Article (Artículo):

Define un contenido independiente que tiene sentido por sí mismo, como un artículo de noticias, un blog o un comentario.

Ejemplo:

    Un post individual en un blog o un artículo en una página de noticias.
